## 🏗️ About

A handcrafted, single-page portfolio built with **zero frameworks** — just HTML, CSS, and vanilla JS. Features a bento-grid layout, GSAP-driven animations, an interactive 3D BoxBot companion (Three.js), and an in-page Snake game. Optimized for performance, SEO, and accessibility.

## 📂 Structure

```
portfolio/
├── index.html            # Main single-page app (6k+ lines)
├── 404.html              # Custom 404 page
├── sw.js                 # Service worker (PWA + caching)
├── robots.txt            # Crawler directives
├── sitemap.xml           # Sitemap for search engines
├── vercel.json           # Vercel routing & headers
├── resume/
│   └── index.html        # Dedicated resume page
├── support/
│   └── index.html        # Support / contact page
└── assets/
    ├── profile.jpg       # Profile photo
    ├── og.png            # Open Graph preview
    ├── covers/           # Project demo videos
    └── favicon/          # Favicons & webmanifest
```
